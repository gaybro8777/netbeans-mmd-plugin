Scia Reto editor

Project

The first step is either select or create a project. Because initially the editor was developed for software projects, "project" means just a folder. You can create new project through File->New project or choose existing folder through File->Open project. Opened project will be remembered and it will be opened automatically during next start of the application. Opened project will be shown in the project tree panel. You can close opened project in any moment, "close" means just removing project from panel without any deletion of the folder on disk.

Files

Main work item in the editor is a file, a file can be created in a project folder through pop-up menu, just activate pop-up menu for a project folder and select needed file format in New.... If you want open a file then just click twice on needed file in project tree. If you want open the file in associated system browser then activate pop-up menu for the file and select Open in System. File can be also removed through Delete and in the case the file will be removed from disk.

Mind maps

Mind map file (with *.mmd extension) will be opened in graphical view and by default it has only one root topic node. If you want add new nodes then:

  • Select mind node with mouse click
  • Activate pop-up
  • Click Add child
  • Also you can add new node by TAB

Pop-up menu is the main way for manipulations with a mind map but also there are some keyboard shortcuts, the main keyboard shortcuts are TAB(to add new child node) and ENTER(to add sibling node), you can navigate between nodes with cursor keys.

PlantUML files

There is embedded editor of PlantUML files, but before you have to install Graphviz graph visualization software.

By default any plantUML editor is opened in view mode, if you want edit plantUML script then you should open script editor panel in the top part of the page, you can make that through either pressing button Edit script or just drag the top page border by mouse.

During script edit operations, the view is updated automatically (if Auto refresh is on) after some time delay, the view is refreshed automatically also just after document save.

F.A.Q.

How to make multiline text in mind map nodes?

By default ENTER stops node text edit but if press SHIFT+ENTER then next line will be added.

How quickly add link to a file?

Mind map supports drag-n-drop operations so that just drag file from project tree by mouse and drop it over needed mind map node.

How quickly create jump between nodes?

Press CNTRL and drag target node by mouse, drop it over node where jump should be presented.

How quickly hide/show project tree panel?

Mouse double click on editor tab allows quickly hide/show project tree panel.

How to select render quality?

You can choose render quality through Edit->Preferences->Render quality

Where edit the list of keyboard shortcuts?

Editable list of keyboard shortcuts is available through Edit->Preferences->Edit keyboard shortcuts

Added images is rescaled, why?

Added images are saved inside text document as BASE64 encoded PNG images, it is very easy to add huge image and kill any possibility to process document, so that by default image is rescaled to have width and height no more than 350 px, It is possible to change the value through Java system property mmap.max.image.side.size
